NetSalary

Este proyecto fue desarrollado con el único objetivo de poder repasar determinadas cuestiones ligadas al lenguaje de programación JavaScript.

Podrás ver, en caso que revises el código, que el mismo no se encuentra comentado, dado que por cuestiones de tiempo y con el objetivo de poder priorizar propiamente la práctica, se omitió realizarlo (aún entendiendo que esto es altamente recomendable).

En caso que el proyecto posea actualizaciones, a futuro se añadirán los respectivos comentarios y guías para poder efectivizar un mejor trabajo colaborativo.

(!) Aclaración: La app es versión de escritorio, no posee versión responsive mobile / tablets.

----------

Lenguajes utilizados:

- JavaScript.
- HTML.
- CSS.
